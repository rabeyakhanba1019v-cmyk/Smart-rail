# Smart Rail Exchange

A secure peer-to-peer railway ticket resale platform that prevents fraud, scalping, and double-selling through trust scores, escrow transactions, and real-time communication.

## Features
- User Authentication
- Ticket Marketplace
- Real-time Chat
- Trust Score System
- Fraud Reporting
- Escrow-based Transactions
- Admin Dashboard

## Tech Stack
- React
- TypeScript
- Tailwind CSS
- Supabase
- PostgreSQL
